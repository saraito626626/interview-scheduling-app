# DB設計

## 5-1. テーブル一覧

| No | テーブル名 | 説明 |
|----|------------|------|
| 1  | エージェント | 面接日程を調整する人材紹介会社の担当者 |
| 2  | 企業 | 面接を実施する企業 |
| 3  | 求職者 | 面接を受ける候補者 |
| 4  | 面接 | 面接情報を管理する |
| 5  | 候補日 | 面接の候補日時を管理する |
| 6  | 回答 | 候補日に対する参加可否の回答 |


### エージェント
| カラム| 型| 説明 |
|----|------------|------|
|エージェントID|INT|エージェントを識別するID|
|氏名|VARCHAR|エージェントの名前|
|メールアドレス|VARCHAR|連絡用メールアドレス|
|作成日|DATETIME|登録した日時|


###企業
| カラム| 型| 説明 |
|----|------------|------|
|企業ID|INT|企業を識別するID|
|企業名|VARCHAR|企業名|
|担当者名|VARCHAR|面接担当者の名前|
|メールアドレス|VARCHAR|連絡用メールアドレス|
|作成日|DATETIME|登録した日時|

###求職者
| カラム| 型| 説明 |
|----|------------|------|
|求職者ID|INT|求職者を識別するID|
|氏名|VARCHAR|求職者の名前|
|メールアドレス|VARCHAR|連絡用メールアドレス|
|作成日|DATETIME|登録した日時|

### 面接
| カラム| 型| 説明 |
|----|------------|------|
|面接ID|INT|求職者を識別するID|
|エージェントID|INT|エージェントを識別するID|
|企業ID|INT|企業を識別するID|
|求職者ID|INT|求職者を識別するID|
|ステータス|VARCHAR|調整中／確定など|
|作成日|DATETIME|登録した日時|

### 候補日
| カラム| 型| 説明 |
|----|------------|------|
|候補日ID|INT|候補日を識別するID|
|面接ID|INT|面接を識別するID|
|候補日|DATETIME|面接候補日時|

### 回答
| カラム| 型| 説明 |
|----|------------|------|
|回答ID|INT|回答を識別するID|
|候補日|DATETIME|面接候補日時|
|回答者種別|VARCHAR|企業/候補者|
|可否|BOOLEAN|面接可否|
